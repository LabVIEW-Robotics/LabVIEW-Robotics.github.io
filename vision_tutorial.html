Vision Tutorial

This page will show how to do standard image processing with NI Vision Assistant. It is assumed that you have gone through the Vision tutorial here and already know how to open the Vision Assistant. The topics covered here are: histogram equalization, noise reduction, canny edge detection, image differencing, and range detection.

Aquiring a Canned Image

For this tutorial we will only be processing a single image from a file or a series of images from a folder. The process for aquiring images in this way is slightly different from using a live feed from a video camera.

  1. Drop the Vision Aquisition express VI onto the block diagram.
  2. In the Vision Aquisition dialog window select Folder of Images.

  3. Click Next
  4. Leave the selection on Single Aquisition with Processing and click Next.
  5. Click the folder icon and navigate to the image you want to process.

  6. Click Finish.

Histogram Equalization

  1. Convert the RGB 32 bit image to a binary 8 bit image. Do this by using the color plane extraction feature of Vision Assistant. Under the color tab, select Color Plane Extraction.

    Then choose the plane to extract. For this tutorial we will chose the RGB - Red Plane.
  2. Now that we have a grayscale image. We do the equalization. Under the Grayscale tab, select Lookup Table.

    Then choose Equalize for the Lookup Table Setup.

Noise Reduction

  1. Convert the image to grayscale if necessary.
  2. Under the Grayscale tab select Filters.

    The choose the type of filter you want to apply. For this tutorial we will choose Smoothing - Low Pass.

Canny Edge Detection

  1. Convert the image to grayscale if necessary.
  2. Under the Grayscale tab select Filters. Scroll down and select Edge Detection - Canny.

Image Differencing

Use color operators to do image differencing in Vision Assistant. To do this you will use image buffers so that you can have two images loaded into Vision Assistant.

  1. Under the Image tab select Get Image. Navigate to your image and select it.
  2. Under the Image tab select Image Buffer. Select Buffer #1. Select OK.
  3. Use the Get Image and Image Buffer options again to load a second image.

  4. Under the Color tab select Color Operators.
  5. Choose the difference operation you wish to apply. For this tutorial we will choose subtraction.
