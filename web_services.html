Web Services

This tutorial will walk you through setting up a web page to display data from LabVIEW. This tutorial uses a computer, but using a myRIO should be very similar.

Software

The project for this tutorial can be downloaded here. When set up and running correctly, the project will display a webpage that displays images from your webcam.

The project includes a computer target this target has a WebService.

To run your website, right click the WebService choose Application Web Server -> Publish. Open a browser goto localhost:8080/WebService.
