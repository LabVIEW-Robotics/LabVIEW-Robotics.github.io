Frequently Asked Questions

This page consists of frequent questions from students.

How do I connect the myRIO to another device on the network?

Communicating with another network device depends on the protocol being used. Fortunately LabVIEW supports many different protocols. To access these protocols, look under the data communication section of the block diagram functions.

A tutorial on using the TCP protocol in LabVIEW can be found on the National Instruments website here. Similarly an tutorial using UDP can be found here.

What is the difference between running the VI on the host machine and the myRIO?

Running a VI on the myRIO gives the VI access to the myRIO's ports. While running it on the host machine will give it access to the host machine ports. For example, if you want to read from a camera connected to the USB port on the myRIO you would want to run the VI on the myRIO. If the USB camera is connected to a USB port on the host machine then you would want to run it on the host machine.

Specifying if the VI will run on the host machine or the myRIO is done in the project view window. Right click on the target and select new -> VI. For more information you can also watch the instructional videos here. While these videos were made for the cRIO they are also applicable to the myRIO.

How do I remove the lag from the camera output?

Communicating with the myRIO over the network will introduce a bandwidth constraint. This causes high definition video output to lag. One way to reduce this lag is to lower the resolution of the video aquisition. This can be done in the vision aquistion wizard on the Configure Aquisition Settings pane. Change the Video Mode setting to a lower resolution.
