Machine Learning

This tutorial will show how to do supervised learning with LabVIEW using the Vision Assistant to detect an object and track the object. The robot will drive towards the object. We will use the iRobot Create and the myRIO. Setup time is about 15 minutes.

Getting Started

  1. Make sure you are connected to the myRIO
  2. Make sure a webcam is connected to the myRIO
  3. Open Vision Assistant from Start Menu.
  4. Select Aquire Image

  5. Choose USB camera

  6. Change the target to the myRIO

  7. Click the play button to see video from the camera

  8. While the camera is playing move the object into view and click the aquire button

  9. Under Acquire Image select the Close button.
  10. In the top right of the window go to the Process Images tab

  11. Under Machine Vision choose Object Tracking

  12. Choose New Object Tracking File

  13. Draw a box around the object to train.
  14. Adjust the number of histogram bin for the best contrast and choose add

The LabVIEW community has created a toolkit to implement machine learning in LabVIEW. This page will provide a breif description on how to get started using the toolkit and running an example VI.
