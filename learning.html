Machine Learning

This tutorial will show how to do supervised learning with LabVIEW using the Vision Assistant to detect an object and track the object. The robot will drive towards the object. We will use the iRobot Create and the myRIO. Setup time is about 15 minutes.

Getting Started

  1. Make sure you are connected to the myRIO
  2. Make sure a webcam is connected to the myRIO
  3. Open Vision Assistant from Start Menu.
  4. Select Aquire Image

  5. Choose USB camera

  6. Change the target to the myRIO

  7. Click the play button to see video from the camera

  8. While the camera is playing move the object into view and click the aquire button

  9. Under Acquire Image select the Close button.
  10. In the top right of the window go to the Process Images tab

  11. Under Machine Vision choose Object Tracking

  12. Choose New Object Tracking File

  13. Draw a box around the object to train.
  14. Adjust the number of histogram bin for the best contrast and choose add

  15. Choose a label and select OK

  16. Save the file to your desktop
  17. We will use this file later
  18. Close the Vision Assistant
  19. Download the project here
  20. Open the project file
  21. Open the vision.vi

  22. Go to the block diagram
  23. Look over the block diagram to understand it. Do this for a few minutes. Ask questions if needed.
  24. Double click the Vision Acquisition, test the myRIO camera feed.
  25. When done testing choose finish
  26. Double click the Vision Assistant
  27. Double click the Object Tracking in the script section

  28. Use the browse files button to choose the NI Tracking file you created earlier

  29. Select OK then Finish
  30. Place the robot so that it can see the object
  31. Run the VI, the robot should drive towards the object
  32. Click both the stop buttons on the front panel to stop the robot

Machine Learning Toolkit

The LabVIEW community has created a toolkit to implement machine learning in LabVIEW. This page will provide a breif description on how to get started using the toolkit and running an example VI. Download the machine learning toolkit and install the functions from here. When you have finished the installation process you should have a new category of functions.

Also download the examples and open the SVM example. This is an exemple using support vector machine to classify objects.
